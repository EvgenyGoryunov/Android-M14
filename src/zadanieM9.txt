Задание
Для модерации постов на канал телеграмм бота необходимо сформировать недельный список будущих выходов постов.
Есть 2 файл с шутками разных авторов, также есть 2 файл с фразами мотивации разных авторов.
Требуется:
Сформировать новый файл (output.txt), в который выводится список следующего формата:
1.	Сначала идет разделительная черта
2.	Далее выводится дата определенного формата
3.	Затем заголовок категории (либо ШУТКА, либо МОТИВАЦИЯ)
4.	В каждом подразделе выводится по два произвольных сообщения с нумерацией в начале


Пример ниже
-------------------------------------------------------------------------------------------------
дата: понедельник (14-февраля-2022)
 ШУТКА
1. А кофе на клавиатуру тоже виpyс пpолил?
2. Ребята, с Моникой Левински надо кончать.
 МОТИВАЦИЯ
1. В любом наборе исходных данных самая надежная величина, не требующая никакой проверки, является ошибочной.
2. Если Вас никто не критикует – значит, успеха Вы еще не добились.
